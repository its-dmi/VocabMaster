# VocabMaster
A word learning app made to help you expand your vocabulary easily!

This is a word learner app made to help the user increase their vocabulary using the help of AI from groq ai api service.
The user must use their own api key FROM GROQ.
I have added an automatic updater (even though it was unnecessary for such a simple app), for testing and learning how I can implement an automatic updater, a bootloader, and the main program to seemlessly work together to reduce the user's hassle of looking for an update and then reinstalling everything.
